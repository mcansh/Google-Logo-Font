Using the Google logo font on your website

Google

the font isn't on Google Fonts but it does use the same api and is available using

@import url("https://fonts.googleapis.com/css?family=Product+Sans");

and then you just need to tell css to use the fonts

body { font-family: "Product Sans" }

or simply

@import url("https://mcansh.github.io/google-logo-font/css/google-logo-font.sass")
